Actividad Final Propuesta

Una reconocida compañía de colectivos desea desarrollar un sistema web para administrar y mantener un registro de todos los viajes realizados. El sistema permitirá gestionar información sobre personas, colectivos y viajes.

Requerimientos del sistema:

1. Personas:

  • Crear, modificar y listar pasajeros.
  • Cada pasajero debe tener nombre, apellido y edad.

2. Colectivos:

  • Crear, modificar y listar colectivos.
  • Cada colectivo debe tener patente, cantidad de asientos y modelo. El modelo debe incluir nombre y marca.

3. Viajes:

  • Crear, modificar y listar viajes.
  • Cada viaje debe tener lugar de salida, lugar de destino, colectivo asignado, listado de pasajeros, fecha y hora de salida, y fecha y hora de llegada.

Tareas a realizar:

  1. Diseñar una interfaz de administrador con 3 opciones principales: Personas, Colectivos y Viajes.
  2. Implementar las funcionalidades de creación, modificación y listado para cada opción mencionada.
